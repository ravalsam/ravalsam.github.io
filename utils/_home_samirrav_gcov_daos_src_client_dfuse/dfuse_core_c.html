28 /*
29  * Wrapper function that is called from FUSE to send RPCs. The idea is to
30  * decouple the FUSE implementation from the actual sending of RPCs. The
31  * FUSE callbacks only need to specify the inputs and outputs for the RPC,
32  * without bothering about how RPCs are sent. This function is also intended
33  * for abstracting various other features related to RPCs such as fail-over
34  * and load balance, at the same time preventing code duplication.

45      * Keep a reference on the inode open which will be dropped after
